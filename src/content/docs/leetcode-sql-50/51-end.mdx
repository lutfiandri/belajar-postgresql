### Apa itu HackerRank?

HackerRank adalah platform online buat latihan coding dan skill assessment. Mereka nyediain sertifikasi gratis yang recognized secara global dan bisa jadi nilai plus di CV kamu.

### 3 Level Sertifikasi SQL

HackerRank punya 3 level sertifikasi SQL yang bisa kamu ambil:

1. **SQL (Basic)** - Fundamental SQL queries
2. **SQL (Intermediate)** - JOIN, subqueries, aggregation
3. **SQL (Advanced)** - Window functions, CTE, complex queries

### Cara Ambil Sertifikasi

1. Bikin akun di [HackerRank](https://www.hackerrank.com/)
2. Masuk ke menu **Certify**
3. Pilih sertifikasi SQL yang mau kamu ambil
4. Kerjain test-nya (biasanya 1-2 jam)
5. Kalau lulus, sertifikat langsung bisa kamu add ke LinkedIn!
