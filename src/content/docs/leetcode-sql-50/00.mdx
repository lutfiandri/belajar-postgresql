## Tentang Seri Ini

Seri **LeetCode SQL 50** adalah kumpulan pembahasan soal-soal SQL dari LeetCode yang dirancang khusus untuk membantu kamu belajar PostgreSQL dari dasar hingga menengah. Setiap soal dilengkapi dengan penjelasan step-by-step yang mudah dipahami.

## Format Pembelajaran

Setiap soal dalam seri ini memiliki format:

- **Overview**: Ringkasan singkat tentang soal
- **Soal**: Deskripsi lengkap soal dari LeetCode
- **Setup Soal**: Script SQL untuk membuat tabel dan mengisi data contoh
- **Jawaban SQL**: Solusi akhir yang bisa langsung digunakan
- **Penjelasan**: Penjelasan step-by-step 
- **Referensi**: Daftar syntax dan fungsi SQL yang digunakan dalam soal

## Cara Menggunakan Materi Ini

### 1. Setup Database

Sebelum mulai, pastikan kamu sudah memiliki PostgreSQL terinstall. Jika belum, kamu bisa:

- Install PostgreSQL secara lokal di komputer kamu
- Menggunakan PostgreSQL online seperti [DB Fiddle](https://www.db-fiddle.com/) atau [SQLPad](https://sqlpad.io/)
- Menggunakan Docker untuk menjalankan PostgreSQL

### 2. Ikuti Urutan Soal

Mulai dari soal pertama dan ikuti urutannya. Setiap soal dibangun berdasarkan konsep sebelumnya, jadi lebih baik mengikuti urutan yang ada.


### 3. Gunakan Setup Soal

Copy script dari bagian **Setup Soal** untuk membuat tabel dan mengisi data. Ini memastikan kamu punya data yang sama seperti di contoh.

### 4. Coba Sendiri Dulu

Sebelum melihat jawaban, coba selesaikan soal sendiri terlebih dahulu. Ini akan membantu kamu memahami konsep dengan lebih baik.

### 5. Pelajari Penjelasan

Setelah mencoba sendiri, baca bagian **Penjelasan** untuk memahami konsep dan teknik yang digunakan.

## Prasyarat

Sebelum mulai, pastikan kamu sudah familiar dengan:

- Dasar-dasar SQL (SELECT, FROM, WHERE)
- Konsep database dan tabel
- Tipe data dasar (INT, VARCHAR, DATE, dll)

Jika belum, tidak masalah! Soal-soal awal dirancang untuk pemula dan akan menjelaskan konsep dasar secara bertahap.
